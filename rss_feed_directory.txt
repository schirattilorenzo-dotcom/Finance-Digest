RSS FEED DIRECTORY — Reputable, Free, Globally-Recognized News Sources
========================================================================
Format: Name | Link | Description (max 100 chars)

Notes:
- Reuters is intentionally excluded — Reuters discontinued public RSS
  feeds around 2020, and feeds claiming to be "Reuters RSS" elsewhere
  are usually unofficial Google News proxies, not the real thing.
- Publishers occasionally rename or retire feed URLs without notice.
  If a feed in this list returns no entries, treat it as dead and
  remove it — this is normal maintenance for any list this size.
- This list totals roughly 150 feeds. It intentionally does not reach
  1000 — see the chat explanation for why that number wasn't realistic
  to responsibly verify at "reputable, well-known" quality.
========================================================================
